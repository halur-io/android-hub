# Video Background Instructions

To add a video background to the hero section:

1. Add your video file to this directory (`static/videos/`)
2. Name it `sushi-preparation.mp4` (or update the filename in `src/components/HeroNew.jsx`)

## Recommended Video Specifications:
- Format: MP4 (H.264 codec)
- Resolution: 1920x1080 (Full HD) or higher
- Duration: 15-30 seconds (will loop)
- File size: Under 10MB for optimal loading
- Content suggestions:
  - Sushi chef preparing rolls
  - Asian cuisine preparation
  - Restaurant ambiance footage
  - Food plating sequences

## Free Stock Video Resources:
- Pexels Videos: https://www.pexels.com/videos/
- Pixabay: https://pixabay.com/videos/
- Videvo: https://www.videvo.net/

Search for: "sushi", "asian food", "japanese cuisine", "restaurant kitchen"
