מדריך מלא לחיבור מדפסת SNBC BTP-R880NP

סקירה כללית

מדפסת SNBC BTP-R880NP היא מדפסת תרמית מקצועית למטבחים ומסעדות. המערכת תומכת בהדפסה אוטומטית של הזמנות ישירות למדפסת.

תכונות נתמכות:
  • הדפסה בעברית (RTL)
  • חיתוך אוטומטי
  • רוחב נייר 80mm או 58mm
  • חיבור רשת Ethernet
  • פרוטוקול ESC/POS

1 הכנת המדפסת

חיבור פיזי:

  1. חבר את המדפסת לחשמל - וודא שהמדפסת דולקת (נורית ירוקה)
  2. חבר כבל רשת (Ethernet) - מהמדפסת לראוטר/סוויץ' ברשת
  3. הכנס גליל נייר תרמי - 80mm או 58mm לפי הדגם

דיאגרמת חיבור:

[אתר אינטרנט] ←→ [שרת] ←→ [רשת מקומית] ←→ [מדפסת SNBC]
                              ↑
                           [ראוטר]

2 הגדרת רשת במדפסת

קבלת כתובת IP:

  1. לחץ והחזק את כפתור FEED כשהמדפסת כבויה
  2. הדלק את המדפסת תוך כדי לחיצה
  3. המדפסת תדפיס דף הגדרות עם כתובת ה-IP
דוגמה לכתובת IP שתקבל: IP Address: 192.168.1.100 Port: 9100 MAC: 00:11:22:33:44:55

שינוי כתובת IP (אם נדרש):

  1. גש לממשק הניהול של המדפסת דרך הדפדפן: http://192.168.1.100
  2. היכנס עם: Username: admin, Password: admin
  3. עבור ל-Network Settings
  4. הגדר IP סטטי או DHCP לפי הצורך
חשוב: רשום את כתובת ה-IP - תצטרך אותה בהמשך!

3 הגדרת חומת אש (Firewall)

פתיחת פורט 9100:

המדפסת משתמשת בפורט 9100 לקבלת פקודות הדפסה. יש לוודא שהפורט פתוח:

בראוטר/חומת אש:

Protocol: TCP Port: 9100 Direction: Inbound (כניסה) Source: IP של השרת/אתר Destination: IP של המדפסת

בדיקת חיבור:

# בדיקה מהשרת ping 192.168.1.100 # בדיקת פורט telnet 192.168.1.100 9100

4 הגדרה במערכת

  1. היכנס לפאנל הניהול: /admin
  2. עבור ל-"מדפסות SNBC"
  3. לחץ על "הוסף מדפסת חדשה"
  4. מלא את הפרטים:
שם המדפסת: מטבח ראשי כתובת IP: 192.168.1.100 פורט: 9100 דגם: SNBC BTP-R880NP רוחב נייר: 80mm תמיכה בעברית: ✓ חיתוך אוטומטי: ✓

בדיקת חיבור:

  1. הכנס את כתובת ה-IP
  2. לחץ על "הדפס עמוד בדיקה"
  3. וודא שהמדפסת הדפיסה את עמוד הבדיקה

5 הגדרת הדפסה אוטומטית

הפעלת הדפסה אוטומטית:

  1. עבור ל-"הגדרות מערכת"
  2. בחר בלשונית "מטבח"
  3. סמן "הדפסה אוטומטית של הזמנות"
  4. בחר את המדפסת שהגדרת
כעת כל הזמנה חדשה תודפס אוטומטית!

פתרון בעיות

המדפסת לא מדפיסה:

  • ✓ וודא שהמדפסת דולקת ומחוברת לרשת
  • ✓ בדוק שכתובת ה-IP נכונה
  • ✓ וודא שפורט 9100 פתוח
  • ✓ בדוק שיש נייר במדפסת

הדפסה לא קריאה:

  • ✓ נקה את ראש ההדפסה
  • ✓ החלף גליל נייר (נייר תרמי בלבד!)
  • ✓ בדוק הגדרות חום בממשק המדפסת

עברית לא מודפסת נכון:

  • ✓ וודא שתמיכת RTL מופעלת בהגדרות
  • ✓ בדוק קידוד UTF-8

פרטים טכניים

פקודות ESC/POS נתמכות:

ESC @ - Initialize printer ESC a - Text alignment (0=left, 1=center, 2=right) ESC ! - Font size GS V - Paper cut GS k - Barcode printing ESC t - Character code table (Hebrew: CP1255)

מפרט טכני:

  • מהירות הדפסה: 300mm/sec
  • רזולוציה: 203 DPI
  • רוחב הדפסה: 72mm (80mm paper) / 48mm (58mm paper)
  • חיבורים: Ethernet, USB, Serial
  • פרוטוקול: ESC/POS compatible
