Goal:
Add a new “Catering & Special Events” section to my existing restaurant website hosted on Replit.
The website is for an Asian restaurant, and this new section should match the style, width, and overall layout of the existing site.

🔧 Requirements

Section Placement

Add the new section below the existing main sections (e.g., after “Menu” or “About”).

Maintain the same full-width layout as other sections.

The new section should be visually balanced — not taller or narrower than others.

Design & Styling

Use an Asian-inspired background image (e.g., elegant food spread, event setup, or minimal texture).

The background image should:

Fit section width (responsive, background-size: cover).

Use an overlay (dark gradient or semi-transparent color) to ensure text readability.

Maintain consistent typography, padding, and colors with the rest of the site.

Content

Add a title: “Catering & Special Events”

Subtitle / short description (SEO-friendly):

“Bring authentic Asian flavors to your next celebration. We offer custom catering menus for weddings, birthdays, and corporate events.”

Include a call-to-action button:

Text: “Request a Quote”

Link: Anchor to contact form (e.g., #contact) or open a new form modal.

Optionally display 3 highlights or icons (e.g., “Custom Menus”, “On-site Service”, “Private Dining”).

SEO Optimization

Use proper semantic tags (<section>, <h2>, <p>, <a>).

Add relevant keywords: “Asian catering”, “event catering”, “special events”, “restaurant catering”.

Ensure image uses alt text and descriptive filename.

Add structured meta description snippet if applicable.

Mobile Responsiveness

Ensure layout adapts gracefully on small screens.

Text remains readable (adjust font sizes, spacing).

Button remains easily tappable.

Background image scales correctly on mobile (object-fit: cover or background-size: cover).

Implementation Details

If site uses HTML + CSS: Add directly to index.html and stylesheet.

If using React or Flask with templates: Add the section as a reusable component or template block.

Test that the section integrates seamlessly with the navigation (scroll or anchor link).

⚙️ Output Expectation

Generate all required HTML, CSS, and image placeholder reference.

Include comments for where to replace the image later.

Code should be SEO-friendly, mobile-responsive, and consistent with the site’s design.
