מאפייני נגישות באתר

  • ניווט מקלדת: ניתן לנווט באתר באמצעות מקש Tab וחצים בלבד
  • תאימות לקוראי מסך: האתר תואם לקוראי מסך כגון JAWS, NVDA
  • ניגודיות צבעים: הצבעים באתר עומדים ביחס ניגודיות מינימלי של 4.5:1
  • שינוי גודל טקסט: ניתן להגדיל טקסט עד 200% ללא אובדן תוכן
  • טקסט חלופי לתמונות: כל התמונות כוללות תיאור טקסטואלי
  • כותרות ומבנה: השימוש בכותרות (H1-H6) הוא הירכי וסמנטי
  • תמיכה ב-RTL: תמיכה מלאה בכיוון ימין לשמאל לשפה העברית

Standards and Compliance

This website has been made accessible in accordance with Israeli Standard (IS) 5568 and WCAG 2.1 Level AA guidelines, pursuant to the Equal Rights for Persons with Disabilities Law, 1998.

Accessibility Features

  • Keyboard Navigation: Full website navigation using Tab and arrow keys
  • Screen Reader Compatible: Compatible with JAWS, NVDA screen readers
  • Color Contrast: Colors meet minimum 4.5:1 contrast ratio
  • Text Resizing: Text can be enlarged up to 200% without loss of content
  • Alternative Text: All images include descriptive text
  • Headings Structure: Hierarchical and semantic heading usage (H1-H6)
  • RTL Support: Full right-to-left support for Hebrew language
